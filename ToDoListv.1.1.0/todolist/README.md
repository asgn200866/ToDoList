# TDL v1.1.0

## Верхняя часть (шапка)

- Название программы => Левый верхний угол.
- Текущая дата => Справа от названия. Формат: "ДД.ММ.ГГГГ".
- Кнопка "Добавить задачу" => Правый верхний угол. При нажатии создаётся новая задача и открывается её окно (детальный просмотр). Задачи могут иметь одинаковые имя, время, тип и другие данные.
  - Дата/Время => Автоматически устанавливается дата на завтрашний день и чекбокс "Весь день", но данные доступны для редактирования.
  - Название задачи => Обязательно для заполнения.
  - Тип задачи => Обязательно для заполнения.
  - Комментарий
  - Кнопка "Создать" => Вызывает окно подтверждения действия.
    - Кнопка "Да" => Надпись "Создание задачи" (Окно выполнения операции).
    - Кнопка "Нет" => Закрытие окна подтверждения.
  - Кнопка "Отменить" => Вызывает окно подтверждения действия.
    - Кнопка "Да" => Надпись "Отмена" (Окно выполнения операции).
    - Кнопка "Нет" => Закрытие окна подтверждения.
- Кнопка "Архив"

## Главное окно

- Содержит колонки, отображающие задачи (краткий вид). Нажатие на задачу откроет её детальный просмотр. Задачи отображаются по сроку выполнения (от раннего к позднему).

- Просроченные задачи
- Задачи на сегодня
- Задачи на завтра
- Задачи на будущее => Содержит разделы в зависимости от срока. Отображаются задачи с сроком позже завтрашнего дня.
  - Задачи на текущую неделю
  - Задачи на следующую неделю
  - Задачи на месяц
- Выполненные задачи => Отображение происходит от самой новой к самой старой.

## Задача (краткий вид)

- Время
- Название задачи
- Тип задачи
- Комментарий => Первые N символов.

## Задача (детальный просмотр)

- Дата/Время
- Название задачи
- Тип задачи
- Комментарий
- Кнопка "Начать" => Надпись "Начало выполнения" (Окно выполнения операции). Краткий вид задачи цвет для активных задач.
- Кнопка "Закончить" => Надпись "Завершение выполнения" (Окно выполнения операции). Краткий вид приобретает цвет выполненных задач.
- Кнопка "Удалить" => Вызывает окно подтверждения действия.
  - Кнопка "Да" => Надпись "Удаление" (Окно выполнения операции).
  - Кнопка "Нет" => Закрытие окна подтверждения.
- Кнопка "Редактировать" => По нажатию открывает доступ к редактированию данных.
  - Кнопка "Сохранить" => Вызывает окно подтверждения действия.
    - Кнопка "Да" => Надпись "Сохранение" (Окно выполнения операции).
    - Кнопка "Нет" => Закрытие окна подтверждения.
  - Кнопка "Отменить" => Закрывает доступ к редактированию без сохранения изменений.

* Выполненная задача получает статус "Выполнена" и может быть перемещена в архив, перемещается в колонку "Выполненные задачи" и не поддаётся редактированию. Кнопка "Редактировать" блокируется.
  - Кнопка "В архив" => Вызывает окно подтверждения действия.
    - Предупреждение => "Задача не может быть возвращена из архива или отредактирована.".
    - Кнопка "Да" => Надпись "Архивирование" (Окно выполнения операции).
    - Кнопка "Нет" => Закрытие окна подтверждения.

## Календарь (Дата/Время)

- Показывает текущий месяц в таблице 6 строк на 7 столбцов. Столбцы содержат названия дней недели в сокращенном виде. Пустые ячейки в днях недели заполняются числами предыдущего и следующего месяца. Название месяца указывается над блоком дней. Справа и слева кнопки стрелок для переключения на следующий/предыдущий месяц. Календарь остаётся статичным при выборе времени.
- Время располагается правее календаря, примыкая к нему. Отображает вертикальную шкалу времени. Прокруткой колеса мыши прокручивается шкала времени. При нажатии на поле выбирается указанный час и сбрасывается чекбокс "Весь день". Зажатие мыши создаёт интервал от начального до конечного времени и и отображается затемнением первого поля времени и всех полей, затронутых интервалом.
- Чекбокс "Весь день". Закреплен в шкале времени над полями и не поддается прокрутке.

## Окно выполнения операции

- Фон темнеет. Открывается окно с соответствующей надписью, обозначающей процесс. Имитирует выполнение фоновой операции. Окно закрывается минимум через одну секунду или по завершению процессов, если операция занимает больше секунды.

## Архив

- Показывает списки задач в кратком осмотре. Нажатие на задачу откроет её детальный просмотр. Архивировать можно только завершённую задачу через детальный просмотр (Кнопка "В архив"). Задачи в архиве не поддаются редактированию и не могут быть возвращены в главное окно. Задачи в архиве автоматически удаляются через месяц от срока планируемого выполнения. Возможно предварительное удаление.
- В левом верхнем углу располагается кнопка "Назад", возвращающая на главный экран.
- Справа от кнопки "Назад" располагаются инструменты поиска и фильтрации задач для просмотра.
  - Поле ввода "Поиск", для фильтрации задач по имени.
  - Выпадающий список типов задач. Автоматически пополняется из созданных пользователем типов.
  - Календарь, позволяющий выбрать интервал, в котором было запланировано выполнение задачи.
