Основной экран содержит название программы, в левом верхнем углу, актуальную дату на сегодня правее от названия.

Ниже 4 колонки: просроченные задачи, задачи на сегодня, задачи на завтра, задачи на будущее. Последняя колонка содержит 3 подраздела: задачи на неделю, задачи на следующую неделю, задачи на месяц.

В колонке окно задачи содержит поверхностную информацию: время, имя задачи, тип, комментарий (первые несколько слов). При нажатии на окно задачи она открывается на центре экрана, фон темнеет). В открывшемся окне отображается вся информация из карточки, но: подробнее раскрывается комментарий (полностью), отображается три кнопки: "начать", "закончить", "удалить". Кнопка "удалить" вызывает окно подтверждения. При нажатии начала/окончания появляются соответствующие надписи, выполняющие роль экрана загрузки. У законченных задачей появляется кнопка "убрать в архив".

В правом верхнем углу, по размерам надписей, находятся кнопки: "добавить" задачу", "архив". По нажатию "добавить задачу" появляется пустое, редактируемое окно задачи. Обязательными полями для заполнения являются: дата, имя, тип. Дата автоматически ставится на сегодня. В режимах времени, если не выбрать определенное выставляется "Весь день". Две кнопки: "Добавить", "Отменить". Клавиши требует подтверждения отдельным окном.

Окна выбора даты: небольшой календарь, показывающий текущий месяц в таблице 7х4. Пробелы по количеству дней заполняются предыдущим месяцем. В верхней части окна указывается название текущего месяца и две стрелки в право и в лево, позволяющие переключиться на другой месяц. Время указывается правее от него в виде вертикальной шкалы. При нажатии выбирается определенный час, при зажатии можно увеличить интервал.
